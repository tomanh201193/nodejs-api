## Skeleton for Node.js applications written in TypeScript

### First run
```
npm uninstall typescript
npm install -g typescript
npm uninstall tsc
npm run tsc -- --all
```
### Development

```bash
npm run dev
```

### Running tests

```bash
npm test
```

### Linting

```bash
npm run lint
```

### Building a container

```bash
docker build .
```
